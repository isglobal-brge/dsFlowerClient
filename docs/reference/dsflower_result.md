# Create a dsflower_result object

Create a dsflower_result object

## Usage

``` r
dsflower_result(per_site, pooled = NULL, meta = list())
```

## Arguments

- per_site:

  Named list mapping server names to their raw results.

- pooled:

  NULL (default) or a single aggregated result.

- meta:

  Named list of metadata.

## Value

A `dsflower_result` object.
